# v1.5.3 — All 22 locales fully translated

> **Language / Idioma:** English first, Português (Brasil) below.

Release date: 2026-07-12

## Highlights

- Every shipped locale is now a complete translation. The 20 extended locales (es, zh, hi, ar, bn, fr, de, ja, ru, ko, id, tr, vi, it, pl, nl, fa, th, ur, ms) previously shipped as partial stubs relying on English fallback now carry the full 179-key translation matching `en.js`, with all existing correct strings preserved.
- The i18n checker now enforces strict key parity for **all** locales (previously only `pt_BR`), so any future drop in translation coverage fails CI instead of silently falling back to English.

## Compatibility

- Requires DankMaterialShell `>= 1.4.0`.
- Requires `adguardvpn-cli` in `PATH` or a configured binary path.
- All 22 locales (`en`, `pt_BR` + 20 extended) now provide every translation key; no English fallback is expected for shipped strings.
- No settings migration is required.

## Validation

- All 31 parser tests pass.
- i18n key validation passes with strict parity for all 22 locales.
- Markdown and QML syntax validation pass.
