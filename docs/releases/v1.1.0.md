## AdGuard VPN Plugin v1.1.0

This release focuses on production stability and contributor readiness.

### Highlights

- Added multilingual UI support with locale bundles (`en_US`, `pt_BR`).
- Added translation contribution workflow and documentation.
- Fixed plugin enable failures in DMS caused by i18n singleton wiring.
- Fixed DNS input focus race while editing settings in the widget popout.
- Added registry-ready screenshot and publishing refinements.

### Compatibility

- Requires DankMaterialShell `>= 1.4.0`.
- Requires `adguardvpn-cli` available in PATH (or configured binary path).
