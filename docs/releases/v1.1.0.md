# AdGuard VPN Plugin — v1.1.0

> **Language / Idioma:** English is primary. Portuguese (Brazil) translation follows. _Released 2026-02-26_ · Focus: **production stability & contributor readiness**.

---

## Highlights

| Area | What changed |
| --- | --- |
| **i18n** | Multilingual UI with locale bundles (`en_US`, `pt_BR`). Full translation workflow documented. |
| **Stability** | Fixed plugin-enable failure caused by `AdGuardVpnI18n.qml` singleton wiring. |
| **UX** | Fixed DNS input focus race during settings editing. |
| **Publishing** | Registry-ready screenshots and documentation polish. |

---

## Compatibility

| Requirement | Version |
| --- | --- |
| DankMaterialShell | `>= 1.4.0` |
| `adguardvpn-cli` | Any recent version, available in `$PATH` or custom binary path |
