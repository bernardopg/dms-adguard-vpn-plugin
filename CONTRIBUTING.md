# Contributing

## Translations

Localization is community-driven.

1. Read [i18n/README.md](./i18n/README.md).
2. Add or update locale entries.
3. Keep keys stable and placeholders unchanged (`{location}`, `{mode}`, etc.).
4. Open a pull request.

## Development cycle

```bash
dms ipc plugins reload adguardVPplugin
dms ipc plugins status adguardVPplugin
```
